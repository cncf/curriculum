# Certified GitOps Associate (CGOA) Curriculum

## Exam Description

The **CGOA** certification is designed to validate a candidate’s understanding of GitOps principles, practices, and tooling in cloud native environments.  
It demonstrates knowledge of GitOps fundamentals, including declarative configuration, version control, continuous delivery, automation, observability, and security considerations in a GitOps workflow.

The certification is intended for developers, DevOps engineers, platform engineers, site reliability engineers, and other IT professionals who want to validate their foundational knowledge of GitOps.  
More information about the exam can be found on the [Linux Foundation Training website](https://training.linuxfoundation.org/certification/certified-gitops-associate-cgoa/).
