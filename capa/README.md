# Certified Argo Project Associate (CAPA) Curriculum

## Exam Description

The **CAPA** certification is designed to validate a candidate’s understanding of the Argo project ecosystem and its role in cloud native environments.  
It demonstrates knowledge of core Argo components and their use cases, including Argo Workflows, Argo CD, Argo Rollouts, and Argo Events, as well as GitOps principles and best practices for managing Kubernetes applications.

The certification is intended for developers, DevOps engineers, platform engineers, and IT professionals who want to validate their foundational knowledge of the Argo ecosystem and GitOps workflows.  
More information about the exam can be found on the [Linux Foundation Training website](https://training.linuxfoundation.org/certification/certified-argo-project-associate-capa/).
