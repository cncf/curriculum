# Certified Kubernetes Security Specialist (CKS) Curriculum

## Exam Description

The **CKS** certification is designed to ensure that Kubernetes security specialists have the skills, knowledge, and competency to secure container-based applications and Kubernetes platforms during build, deployment, and runtime.  
It demonstrates proficiency in applying best practices for Kubernetes security, including cluster hardening, system and network security, supply chain security, monitoring, and incident response.

The certification is intended for Kubernetes administrators, cloud security engineers, and other IT professionals who want to validate their expertise in securing Kubernetes environments.  
More information about the exam can be found on the [Linux Foundation Training website](https://training.linuxfoundation.org/certification/certified-kubernetes-security-specialist-cks/).

## Domains & Weighting

| Domain                  | Weight |
|-------------------------|--------|
| Cluster Setup           | 10% |
| Cluster Hardening       | 15% |
| System Hardening        | 15% |
| Minimize Microservice Vulnerabilities | 20% |
| Supply Chain Security   | 20% |
| Monitoring, Logging and Runtime Security | 20% |
