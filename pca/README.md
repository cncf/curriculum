# Prometheus Certified Associate (PCA) Curriculum

## Exam Description

The **PCA** certification is designed to ensure that candidates understand the fundamentals of monitoring and observability using Prometheus.  
It demonstrates knowledge of Prometheus architecture, data model, query language (PromQL), metrics collection, alerting, and integration within cloud native environments.

The certification is intended for site reliability engineers, DevOps engineers, developers, and IT professionals who want to validate their skills in using Prometheus for monitoring and observability.  
More information about the exam can be found on the [Linux Foundation Training website](https://training.linuxfoundation.org/certification/prometheus-certified-associate-pca/).

## Domains & Weighting

| Domain                 | Weight |
|------------------------|--------|
| Fundamentals & Concepts | 20% |
| Prometheus Architecture | 20% |
| PromQL                 | 30% |
| Instrumentation        | 10% |
| Alerting & Recording Rules | 10% |
| Ecosystem & Integrations | 10% |
