# Istio Certified Associate (ICA) Curriculum

## Exam Description

The **ICA** certification is designed to validate a candidateâ€™s knowledge of Istio and its role as a service mesh in cloud native environments.  
It demonstrates proficiency in Istio architecture, traffic management, observability, security, and integrating Istio with Kubernetes for microservices communication.

The certification is intended for developers, platform engineers, DevOps engineers, and IT professionals who want to validate their foundational Istio and service mesh skills.  
More information about the exam can be found on the [Linux Foundation Training website](https://training.linuxfoundation.org/certification/istio-certified-associate-ica/).
