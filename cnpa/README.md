# Certified Cloud Native Platform Engineering Associate (CNPA) Curriculum

This repository contains the open-source curriculum for the **Certified Cloud Native Platform Engineering Associate (CNPA)** certification, maintained by CNCF and The Linux Foundation.

## Exam Description

The **CNPA** is designed for engineers and practitioners working with cloud native platforms.  
It demonstrates knowledge of platform engineering fundamentals, including automation, observability, security, continuous delivery, developer experience, and infrastructure provisioning.  
By earning this certification, candidates prove their ability to contribute to building and operating modern internal developer platforms (IDPs) and enabling efficient, secure, and scalable software delivery within cloud native organizations.

The certification is intended for platform engineers, SREs, DevOps professionals, infrastructure architects, and IT practitioners who want to validate their foundational platform engineering skills.  
More information about the exam can be found on the [Linux Foundation Training website](https://training.linuxfoundation.org/certification/certified-cloud-native-platform-engineering-associate-cnpa/).

## Domains & Weighting

| Domain                                     | Weight |
|--------------------------------------------|--------|
| Platform Engineering Core Fundamentals      | 36%    |
| Platform Observability, Security & Conformance | 20% |
| Continuous Delivery & Platform Engineering  | 16%    |
| Platform APIs & Provisioning Infrastructure | 12%    |
| IDPs & Developer Experience                 | 8%     |
| Measuring Your Platform                     | 8%     |
