# Kubernetes and Cloud Native Security Associate (KCSA) Curriculum

## Exam Description

The **KCSA** certification is designed to ensure that candidates understand the fundamental principles of securing container-based applications and cloud native environments.  
It demonstrates knowledge of cloud native security concepts, including Kubernetes security fundamentals, cloud native security tooling, best practices, and the broader security landscape for cloud native technologies.

The certification is intended for students, developers, administrators, and other IT professionals who want to validate their foundational knowledge of security in the Kubernetes and cloud native ecosystem.  
More information about the exam can be found on the [Linux Foundation Training website](https://training.linuxfoundation.org/certification/kubernetes-and-cloud-native-security-associate-kcsa/).
