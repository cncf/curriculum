# Kyverno Certified Associate (KCA) Curriculum

## Exam Description

The **KCA** certification is designed to validate a candidateâ€™s knowledge and proficiency in using Kyverno as a policy engine for Kubernetes.  
It demonstrates expertise in managing policy as code workflows, including Kyverno fundamentals, installation and upgrades, CLI usage, policy writing, application, and management.

The certification is intended for Kubernetes administrators, DevOps and DevSecOps engineers, security analysts, cloud native developers, and compliance professionals aiming to secure and automate Kubernetes environments.  
More information about the exam can be found on the [Linux Foundation Training website](https://training.linuxfoundation.org/certification/kyverno-certified-associate-kca/) :contentReference[oaicite:1]{index=1}.
