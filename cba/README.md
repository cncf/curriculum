# Certified Backstage Associate (CBA) Curriculum

## Exam Description

The **CBA** certification is designed to validate a candidate’s skills and mindset in building and managing internal developer portals using Backstage, an open platform essential to modern platform engineering workflows.  
It demonstrates proficiency in Backstage development workflows, infrastructure setup, catalog management, and plugin customization.

The certification is intended for IT engineers, developers, platform engineers, and other professionals aiming to enhance developer experience and internal tooling at scale.  
More information about the exam can be found on the [Linux Foundation Training website](https://training.linuxfoundation.org/certification/certified-backstage-associate-cba/).

## Domains & Weighting

| Domain                        | Weight |
|-------------------------------|--------|
| Backstage Development Workflow | 24% |
| Backstage Infrastructure       | 22% |
| Backstage Catalog              | 22% |
| Customizing Backstage          | 32% |
